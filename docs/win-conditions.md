# Win Conditions

## Default wins

| Winner | Condition |
| --- | --- |
| **Good** | All living Demons are dead (typically: the Demon is executed or otherwise killed) |
| **Evil** | Exactly **two** living players remain (Travellers usually do not count toward this) |

Dead players still win or lose with their **alignment team**.

## Simultaneous wins

If good and evil would both win at the same moment (e.g. Demon dies leaving two alive),
**good wins**.

## Multiple Demons

If multiple Demons are in play, **all** must die for good to win (unless a script ability says otherwise).

## Trouble Brewing specials

| Character | Extra / alternate condition |
| --- | --- |
| **Saint** | If the Saint is **executed**, good loses immediately (evil wins). Night death of the Saint does **not** do this. |
| **Mayor** | If only **three** living players remain and **no execution** occurs that day, the Mayor’s team wins. Also: Storyteller may redirect night death from Mayor to another player. |
| **Scarlet Woman** | If the Demon dies while **5+** living players remain, Scarlet Woman becomes the Demon — game does **not** end. With **4 or fewer** living, Demon death ends the game normally (good wins). |
| **Imp starpass** | If Imp kills **themself** at night, a living Minion becomes the Imp — game continues. |

## Character-driven endings (other editions — awareness)

Examples (not exhaustive): Klutz wrong pick, Evil Twin living, Vortox no-execution day,
Mastermind extra day, etc. Always check the active script.

## What does **not** end the game

- A Minion dying
- Reaching final three without an execution (unless Mayor applies)
- A player dying at night (unless they were the last Demon without SW backup, etc.)
- Outsider abilities that only hinder good

## Storyteller checklist after each death

1. Did the (last) Demon die?
   - If Scarlet Woman eligible → convert, continue.
   - Else → good wins.
2. Are exactly two non-Traveller players alive with a living Demon? → evil wins.
3. Did an ability force an instant win/loss (Saint, etc.)?
4. Otherwise continue to next phase.
