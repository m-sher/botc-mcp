# Storyteller Signals (Night)

Players keep eyes closed unless woken. Storyteller is silent; uses gestures.

| Meaning | Signal |
| --- | --- |
| **Eyes open** (wake) | Two taps on shoulder or knee |
| **Eyes closed** (sleep) | Cover your own eyes with a hand |
| **Yes** | Nod |
| **No** | Shake head |
| **Good** | Thumbs up (often after pointing at a player) |
| **Evil** | Thumbs down |
| **Number 0** | Circle with thumb & fingers (OK-style / zero) |
| **Number 1, 2, 3…** | That many fingers |
| **This player** | Point at them |
| **This character** | Show character token or point on script sheet |
| **Confirm choice** | Point down at the player you think they meant; wait for nod |

### Info tokens commonly shown

| Token | Meaning |
| --- | --- |
| This is the Demon | Point at Demon for Minions |
| These are your Minions | Point at each Minion for Demon |
| These characters are not in play | Show three good tokens as bluffs |
| You are | Character/alignment change |
| Died today | Undertaker preface |

Demonstrate signals during the rules explanation before the first night.
