# Glossary

| Term | Definition |
| --- | --- |
| **Storyteller** | Moderator who runs the game; not on a team |
| **Grimoire** | Private Storyteller board of all truths |
| **Script / Edition** | The allowed character pool for a game |
| **Townsfolk** | Good character type with helpful abilities |
| **Outsider** | Good character type that complicates good’s game |
| **Minion** | Evil support character |
| **Demon** | Evil primary; must die for good to win (default) |
| **Traveller** | Special mid-game role; public character, hidden alignment |
| **Fabled** | Storyteller-side balance character |
| **Night*** | Ability skips the first night |
| **Drunk** | State or TB Outsider: ability faked / no real power |
| **Poisoned** | State: no ability; may get false info |
| **Register as** | Appear as another type/alignment to detections |
| **Red herring** | Good player who pings as Demon to Fortune Teller |
| **Starpass** | Imp kills self; Minion becomes Imp |
| **Ghost vote** | Dead player’s single remaining vote |
| **Execution** | Day elimination procedure (≤1 per day) |
| **Exile** | Traveller removal (≠ execution) |
| **Shroud** | Grimoire marker for dead |
| **Bluff** | Character an evil player pretends to be |
| **Not-in-play bluffs** | Three good characters shown to Demon N1 |
| **Town Square** | Public life/vote token board |
| **Teensyville** | Guidance for very small player counts |
