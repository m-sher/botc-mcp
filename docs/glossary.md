# Glossary

| Term | Definition |
| --- | --- |
| **Storyteller** | Moderator / server; not on a team |
| **Grimoire** | Private full game state |
| **Character pool** | Characters that might be in the bag |
| **Townsfolk** | Good type with helpful abilities |
| **Outsider** | Good type that hinders or confuses good |
| **Minion** | Evil support |
| **Demon / Imp** | Evil killer; good wins if they die (unless SW converts) |
| **Night*** | Ability skips the first night |
| **Drunk** | State or Outsider: no real ability / false info |
| **Poisoned** | State: no ability; may get false info |
| **Register as** | Appear as another type/alignment to detections |
| **Red herring** | Good player who pings as Demon to Fortune Teller |
| **Starpass** | Imp kills self; a Minion becomes Imp |
| **Ghost vote** | Dead player’s single remaining vote |
| **Execution** | Day elimination (≤1 per day) |
| **Bluff** | Character an evil player pretends to be |
| **Not-in-play bluffs** | Three good characters shown to Demon on N1 (7+) |
