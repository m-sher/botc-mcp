# The Storyteller

The Storyteller is a full player role without a character token. They:

- Run setup, nights, days, nominations, and end-of-game checks
- Own the **Grimoire** (sole complete truth)
- Deliver ability information (true or deliberately false when allowed)
- Keep the game paced and fun

## Neutrality and discretion

The Storyteller is **not** on good or evil. Discretion includes:

| Allowed | Not allowed |
| --- | --- |
| False info to drunk/poisoned | Breaking printed rules arbitrarily |
| Choosing who is red herring | Secretly running no Demon |
| Mayor night-death bounce target | Adding random Drunk mid-game without an ability |
| Spy/Recluse registration choices | Revealing Grimoire contents |
| Balancing **fun** via legal choices | Ignoring win conditions |

When giving false info, prefer choices that create interesting bluffs and support
believable evil narratives without railroading a predetermined winner.

## Public vs private answers

- Rules questions: answer truthfully (even to drunk/poisoned).
- Strategic advice: allowed and encouraged for new players; prefer private.
- “What do I learn?”: only what their ability grants.

## Running nominations

- Repeat nominations loudly
- Count votes clearly clockwise
- State thresholds and ties
- Last call before execution

## Common TB reminder tokens

- Poisoned
- Drunk / Is the Drunk
- Red herring
- Wrong / Townsfolk (Washerwoman pair)
- Protected (Monk)
- Master (Butler)
- Dead shrouds

## First night demon/minion brief (7+)

1. Minions see Demon (and each other if woken as a group).
2. Demon sees Minions + three safe good bluffs not in play.

## After game

Optional: reveal Grimoire, walk through key false info, celebrate both teams.
