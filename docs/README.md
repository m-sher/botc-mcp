# Blood on the Clocktower — Reference Docs

Organized rules and character reference for the **botc-mcp** eval MCP server.
Content is distilled from official and community sources (wiki, rulebook, published guides)
and structured so agents and evaluators can retrieve one topic at a time.

> **Not an official product.** Blood on the Clocktower is © The Pandemonium Institute.
> Ability text is paraphrased for implementation clarity; when in doubt, prefer the
> official [wiki](https://wiki.bloodontheclocktower.com/) and almanacs.

## Retrieval tips for agents

- Need the **day/night cycle** → `gameplay-loop.md`
- Need **who wins when** → `win-conditions.md`
- Need **vote thresholds** → `voting-and-nominations.md`
- Need a **specific TB role** → `roles/tb-*.md` (search by character name)
- Need **wake order** → `night-order.md`
- Need **drunk/poison rules** → `states.md`
- Need **setup bag composition** → `setup.md`
