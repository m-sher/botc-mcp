# Scarlet Woman

| | |
| --- | --- |
| **Type** | Minion |
| **Team** | Evil |

**If there are 5 or more players alive when the Demon dies, you become the Demon (Imp).**

- You gain full Imp powers going forward.
- The game does **not** end on that Demon death.
- If the Demon dies with **4 or fewer** alive → good wins; you do not convert.
- Conversion is secret to the town; you are informed privately.

On first night with 7+ players, you learn who the Demon is (and fellow Minions).
