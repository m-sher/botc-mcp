# Poisoner

| | |
| --- | --- |
| **Type** | Minion |
| **Team** | Evil |

**Each night, choose a player: they are poisoned tonight and tomorrow day.**

- Previous poison ends when you act again.
- Poisoned player has no ability; may get false info; is not told.
- Once-per-game abilities attempted while poisoned are spent and fail.
- If you die, your active poison ends immediately.

On first night with 7+ players, you learn who the Demon is (and fellow Minions).
