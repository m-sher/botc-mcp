# Recluse

| | |
| --- | --- |
| **Type** | Outsider |
| **Team** | Good |

**You might register as evil / Minion / Demon, even if dead.**

- You are actually a good Outsider with no active night choice.
- Each time an ability would detect you, the Storyteller chooses how you register.
- Registration can vary per detection in the same night.
- You do not gain evil abilities; you do not join the evil team.
