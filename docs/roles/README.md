# Roles

Each character has **one file**. Load only the player’s role to avoid extra ability text.

| Type | Directory | Team |
| --- | --- | --- |
| Townsfolk | [townsfolk/](townsfolk/) | Good |
| Outsiders | [outsiders/](outsiders/) | Good |
| Minions | [minions/](minions/) | Evil |
| Demons | [demons/](demons/) | Evil |

Type taxonomy (no ability text): [character-types.md](character-types.md).

## Path convention

```
docs/roles/<type>/<kebab-name>.md
```

Examples: `roles/townsfolk/fortune-teller.md`, `roles/minions/scarlet-woman.md`, `roles/demons/imp.md`.
