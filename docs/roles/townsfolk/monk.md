# Monk

| | |
| --- | --- |
| **Type** | Townsfolk |
| **Team** | Good |

**Each night*, choose a player (not yourself): they are safe from the Demon tonight.**

- `*` = not the first night.
- Blocks Demon kill (and other Demon harmful effects that night, if any).
- Does not block execution or non-Demon kills.
- Demon does not get a re-pick if they hit the protected player.
