# Undertaker

| | |
| --- | --- |
| **Type** | Townsfolk |
| **Team** | Good |

**Each night*, you learn which character died by execution today.**

- `*` = not the first night.
- Only **execution** deaths (not night kills).
- If no execution death today: no info / do not wake for info.
- If the Drunk was executed: shown as **Drunk**, not the Townsfolk they thought they were.
