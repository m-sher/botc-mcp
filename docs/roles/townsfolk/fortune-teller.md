# Fortune Teller

| | |
| --- | --- |
| **Type** | Townsfolk |
| **Team** | Good |

**Each night, choose 2 players: you learn if either is a Demon.**

- **Yes** if at least one chosen is a Demon **or** the **red herring**.
- Red herring: one good player selected at setup; registers as Demon **only** to you.
- Can pick self; can pick dead; can be the red herring.
- Does not say which of the two pinged.
