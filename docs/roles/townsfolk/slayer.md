# Slayer

| | |
| --- | --- |
| **Type** | Townsfolk |
| **Team** | Good |

**Once per game, during the day, publicly choose a player: if they are the Demon, they die.**

- Public declaration required.
- Wrong guess: nothing happens (no confirmation you are the Slayer beyond the attempt).
- Correct: Demon dies immediately → check win conditions (including Scarlet Woman).
- Spent even if drunk or poisoned at use (fails and cannot retry).
