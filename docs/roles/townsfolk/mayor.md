# Mayor

| | |
| --- | --- |
| **Type** | Townsfolk |
| **Team** | Good |

**If only 3 players live & no execution occurs, your team wins. If you die at night, the Storyteller may choose another player to die instead.**

- 3-alive no-exec win is for your **team** (normally good).
- Night bounce is optional Storyteller discretion; the group does not learn a bounce happened.
- If the bounce target cannot die to the Demon (e.g. protected), **nobody** may die that night.
