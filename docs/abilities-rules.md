# Ability Resolution Rules

These meta-rules govern all character abilities.

## Authority order

1. Character **almanac** (most precise)
2. Character **token / sheet** text
3. **Night sheet** order (wake/reminder guide only)

## Immediate resolution

Abilities resolve **immediately** when their condition is met.

Examples:

- Demon kills Fortune Teller early in night → FT does not wake later that night.
- Monk protects Empath before Demon attacks Empath → Empath lives.

## Information hygiene

- Only tell a player what **their** ability says they learn.
- Group does **not** learn: who the Scarlet Woman became, whether Slayer hit,
  whether Monk protected, who caused a night death, etc.

## Loss of ability

Ability is lost **immediately** on:

- Death (unless text says “even if dead” / triggers on death)
- Becoming drunk
- Becoming poisoned

Persistent reminder effects from a dead/drunk/poisoned source end
(e.g. Poisoner dies → their poison ends even if text said “until dusk”).

## “Choose” vs Storyteller choice

| Wording | Who decides |
| --- | --- |
| “**choose** a player…” | The character’s player |
| “A player is poisoned…” (no choose) | Storyteller |

## Illegal choices

If a player chooses illegally (Monk self-target, Chambermaid dead target, etc.):

- Day: tell them verbally
- Night: shake head **no**, prompt re-choose

## Character change mid-game

- Gain new ability immediately; lose old immediately.
- Once-per-game already used on old character: new character’s once-per-game is available.
- First-night-only abilities: if gained later, they function **tonight**.
- Setup brackets `[+2 Outsiders]` etc. do **not** re-fire.

## Night sheet vs triggers

Night sheet reminds the Storyteller whom to wake. Death-triggered abilities
should resolve when the death happens even if the Storyteller is “out of order”
on the sheet — correct as soon as noticed.

## Registration

When an ability detects character type or alignment, apply **registration**
rules (Spy, Recluse, red herring) before answering. See [states.md](states.md).
